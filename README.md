# fast-finger-backend

Url : https://ff-backend-ajk.herokuapp.com/

### About Game
Rest APIs for Backend of Fast Finger:

* SignIn 
* SignUp
* Game Score Updation

## Tech Stack

* NodeJs
* ExpressJs 
* Jwt
* MySql

## Setup

``` npm install ``` \
``` npm start ```

## Learning Outcomes

* Rest API design and development.
* Mysql integration with Node Server.
* JWT auth token integration.
* Custom response messages.
